获取场景访问趋势

URL

http://api.besth5.com/1/analytics/trends

支持格式

JSON

HTTP 请求方式

GET

资源信息

  • 访问权限限制：ip + access_token
  • 请求频次限制：暂无

请求参数

参数字段 类型及范围 说明
from string 统计区间的起始时间，例如 2016-02-16
to string 统计区间的结束时间，例如 2016-02-19

调用样例

GET http://api.besth5.com/1/analytics/trends?from=2016-02-16&to=2016-02-19T23:59:59

返回结果

返回字段说明

返回值字段 字段类型 字段说明
hour object 分小时统计数据
day object 分天统计数据
key_as_string string 字符串形式的时间
key int 时间戳
value int 指定时间段内的访问数

注意事项

时间表示以此时间为起始点的统计单位，用每小时起点表示该小时数据，用每天起点表示该天数据
