获取用户信息

URL

http://api.besth5.com/1/users/:openid

支持格式

JSON

HTTP 请求方式

GET

资源信息

  • 访问权限限制：ip 及 access_token
  • 请求频次限制：暂无

请求参数

参数字段 类型及范围 说明
openid string 微信用户 openid

调用样例

GET http://api.besth5.com/1/users/ozS6eswbZtUoRI-8s19EuoSXoTHo

返回结果

返回字段说明

返回值字段 字段类型 字段说明
openid string 用户的唯一标识
nickname string 用户昵称
country string 国家，如中国为CN
province string 用户个人资料填写的省份
city string 普通用户个人资料填写的城市
groupid int 用户所在的分组ID
headimgurl string 用户头像
language string 用户的语言，简体中文为zh_CN
remark string 公众号运营者对粉丝的备注
sex int 用户的性别，1 为男，2 为女，0 为未知
subscribe int 用户是否订阅该公众号标识, 1 为已订阅，0 为未订阅
subscribe_time string 用户关注时间，为时间戳
unionid string 只有在用户将公众号绑定到微信开放平台帐号后，才会出现该字段

注意事项

部分字段需要该 openid 关注公众号后才能包含。

  • 开始接入
  • 资源列表
    • 场景类
      • 创建监测场景
      • 获取场景列表
      • 编辑场景信息
      • 访问监测场景
    • 用户类
      • 获取用户列表
      • 获取用户信息
      • 获取用户上游
      • 获取用户下游
    • 会话类
      • 获取最新会话列表
      • 获取会话详情
    • 统计类
      • 获取场景统计信息
      • 获取场景访问趋势
      • 获取场景热门统计
      • 获取场景关键用户
