访问场景

获取场景的访问链接。

注：此接口即为场景/站点的访问入口。

URL

http://api.besth5.com/1/s/:id/:channel

HTTP 请求方法

GET

资源信息

  • 访问权限限制：无
  • 请求频次限制：暂无

请求参数

参数字段 必须 类型及范围 说明
id 是 integer 场景 id
channel 是 string 渠道, 符合 /^[0-9A-Za-z]{3}$/

可以在 queryString 中自定义参数，但不可使用保留变量名（openid, state）, 这些参数会被带到活动首页，如 /s/111/099?p1=v1&p2=v2

返回结果

此类请求，服务器会返回状态码 302，跳转到实际微信授权地址

注意事项

修改站点的 scope 字段会影响此处结果

如果 channel 固定,且没有动态参数的需求, 可以把获取到的跳转地址直接放到微信菜单按钮, 图文页的阅读原文上
