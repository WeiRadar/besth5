编辑场景信息

修改单个监测场景的信息。

URL

http://api.besth5.com/1/sites/:id?access_token

支持格式

JSON

HTTP 请求方式

PATCH

资源信息

  • 访问权限限制：access_token
  • 请求频次限制：暂无

请求参数

参数字段 类型及范围 说明
id 是 int id 活动号
scope 是 string snsapi_userinfo: 授权,有用户信息; snsapi_base: 隐式授权, 只有openid
uri 是 string 由于不允许修改 authorizer_appid, 所以也不允许修改首页的域名, 只能修改path
name 否 string 活动名称
originalUri 否 string 原始地址

调用样例

PATCH http://api.besth5.com/1/sites/1?access_token=
