获取最新会话列表

获取最新页面访问记录（会话）。

URL

http://api.besth5.com/1/sessions

支持格式

JSON

HTTP 请求方式

GET

资源信息

  • 访问权限限制：ip + access_token
  • 请求频次限制：暂无

请求参数

参数字段 类型及范围 说明
size int 返回结果的条数，默认为 10，最大为 1000

调用样例

GET http://api.besth5.com/1/sessions?size=3

返回结果

返回字段说明

返回值字段 字段类型 字段说明
total int 总访问人次
code string 访问 id
pcode string 引荐者 id，即此次访问是由之前哪次访问带来的
openid string 微信用户 openid，对当前公众号是唯一的
scope string 授权类型，snsapi_userinfo 或 snsapi_base
ip string 访问者 ip
ua string 访问者 UserAgent
createdAt string 访问时间，东 8 区
appid string 公众号的 appid
ua_data.os object 访问者操作系统信息
ua_data.engine object 访问者浏览器信息
ua_data.micro object 访问者微信版本
ua_data.nettype object 访问者网络类型
ua_data.device object 访问者设备信息
userinfo.openid string 用户的唯一标识
userinfo.nickname string 用户昵称
userinfo.country string 国家，如中国为CN
userinfo.province string 用户个人资料填写的省份
userinfo.city string 普通用户个人资料填写的城市
userinfo.groupid int 用户所在的分组ID
userinfo.headimgurl string 用户头像
userinfo.language string 用户的语言，简体中文为zh_CN
userinfo.remark string 公众号运营者对粉丝的备注
userinfo.sex int 用户的性别，1 为男，2 为女，0 为未知
userinfo.subscribe int 用户是否订阅该公众号标识, 1 为已订阅，0 为未订阅
userinfo.subscribe_time string 用户关注时间，为时间戳
userinfo.unionid string 只有在用户将公众号绑定到微信开放平台帐号后，才会出现该字段

注意事项

用户信息的详细程度取决于实际情况，只有 openid 是一定包含的。

  1. 若 scope 为 snsapi_base，则仅含 openid 信息
  2. 若 scope 为 snsapi_userinfo，则包含用户昵称、性别、头像、地址等信息（详细说明）
  3. 若用户已订阅公众号，则包含分组、备注、订阅时间、语言等信息（详细说明）
