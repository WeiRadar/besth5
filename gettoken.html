获取 access_token

数据接口需要携带 access_token 请求才能成功返回数据，因此需要获取可用 access_token。

URL

http://open.admaster.com.cn/api/oauth/access_token

支持格式

JSON

HTTP 请求方式

POST

资源信息

  • 访问权限限制：无
  • 请求频次限制：暂无

请求参数

参数字段 必选 类型及范围 说明
client_id 是 string 固定值，为 31176d61b449a49d3d76
grant_type 是 string 固定值，为 password
email 是 string 您在 AdMaster 用户中心的邮箱
password 是 string 您在 AdMaster 用户中心的密码

调用样例

POST http://open.admaster.com.cn/api/oauth/access_token

Body(x-www-form-urlencoded)

curl 请求样例

请将 youremail 与 yourpassword 修改为自己的邮箱和密码。
