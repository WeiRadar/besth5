获取场景统计信息

获取场景的综合播统计数据。

URL

http://api.besth5.com/1/analytics

支持格式

JSON

HTTP 请求方式

GET

资源信息

  • 访问权限限制：ip + access_token
  • 请求频次限制：暂无

请求参数

参数字段 类型及范围 说明

调用样例

GET http://api.besth5.com/1/analytics

返回结果

返回字段说明

返回值字段 字段类型 字段说明
total int 总访问次数
user int 总访问人数
kols int 子站点数量
deep int 最大传播深度
from.msg int 从微信消息带来的访问次数
from.tln int 从朋友圈带来的访问次数
from.n int 其他途径带来的访问次数
sex object 访客性别统计
subscribe object 访客订阅公众号情况统计
share object 分享情况统计

注意事项

只有 scope 为 snsapi_userinfo 时，才有访客性别、订阅情况统计。
